# 阿里云盘青龙签到脚本

## 功能

- ✅ 自动每日签到
- 🎁 自动领取签到奖励
- 📊 签到日历 + 奖励详情
- 📤 PushPlus 微信推送通知
- 👥 支持多账号（环境变量中 `&` 或换行分隔）
- 🔄 失败自动重试（3次）
- 🎲 随机 UA + 多账号间隔随机延迟，降低检测风险

## 快速开始

### 1. 青龙面板添加依赖

```
依赖管理 → 新建依赖 → 名称: axios → 类型: NodeJs
```

### 2. 获取 refresh_token

浏览器打开 https://www.aliyundrive.com/ 并登录，然后：

- F12 打开开发者工具
- 切换到 Console（控制台）标签
- 粘贴以下代码并回车：

```js
JSON.parse(localStorage.token).refresh_token
```

复制输出的值（约30-40位字符）。

### 3. 青龙面板添加环境变量

| 名称 | 值 | 备注 |
|------|-----|------|
| `refreshToken` | 你的 refresh_token | 必填。多账号用 `&` 或换行分隔 |
| `ALIYUN_PUSH_PLUS` | PushPlus Token | 可选。用于微信推送通知 |

> **PushPlus Token 获取**：访问 https://www.pushplus.plus/ 登录后复制 Token

### 4. 青龙面板创建定时任务

```
定时任务 → 新建任务
- 名称: 阿里云盘签到
- 命令: task aliyundrive_checkin.js
- 定时规则: 0 8 * * * (每天早上8点，可自定义)
```

### 5. 手动运行测试

点击任务的"运行"按钮，查看日志确认签到成功。

## 常见问题

### refresh_token 过期了怎么办？
按第2步重新获取即可。环境变量中更新新的 refresh_token。

### 签到失败？
- 检查 refresh_token 是否正确（注意复制完整，不要漏字符）
- 检查青龙面板是否能正常访问外网（api.aliyundrive.com、member.aliyundrive.com）

### 多账号如何配置？
环境变量中用 `&` 分隔，例如：
```
token1&token2&token3
```
或换行分隔每个 token。
